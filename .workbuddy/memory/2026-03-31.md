### Railway 数据恢复步骤
1. 重新部署后，进入 Railway Dashboard → 你的服务 → Shell
2. 执行以下命令：
   ```bash
   cd /app
   # 上传 data_backup_20260331.zip 到项目根目录
   unzip data_backup_20260331.zip -d /tmp/backup
   cp -r /tmp/backup/* data/
   ```
3. 重启服务

### 注意事项
- Railway 每次重新部署会丢失容器内数据
- 建议配置 Railway Volume 或使用外部数据库实现持久化
- 免费版 Volume 限制 0.5GB
